Zoom

Utilize os comandos que acionam as funções do zoom

Para aumentar a definição, pressione a tecla CTRL e a tecla + simultaneamente.

Para diminuir a definição, pressione a tecla CTRL e a tecla - ao mesmo tempo.

Para voltar ao tamanho normal, pressione a tecla CTRL e a tecla 0 (zero).

Para voltar na página anterior, pressione a tecla ALT+seta para esquerda.

Ir para próxima página, pressione ALT+seta para direita.
